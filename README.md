## PROJECT 1: Data Jobs Salary Dashboard(Interactive Project)


## Overview
This interactive **Salary Dashboard** was built as the capstone project for Luke Barousse's *Excel for Data Analytics – Full Course for Beginners*.  
It helps job seekers, recruiters, and aspiring data professionals quickly explore real-world salary data for data science, data analyst, data engineer, and machine learning roles (based on 2023 job postings).

The dashboard lets users filter by job title, location (country/region), experience level, employment type, and more — revealing median salaries, salary distributions, and top-paying combinations at a glance.

## Key Features
- **Interactive Slicers** for job title, country, work setting (remote/hybrid/on-site), and experience level
- **Dynamic charts**: Bar charts for average salary by job title, box plots for salary ranges, pie charts for job distribution
- **Key metrics cards**: Median salary, average salary, total job postings, highest/lowest paying roles
- Clean layout with conditional formatting and sparklines for quick insights
- Built entirely in **Excel**

# PROJECT 2: Data Jobs Salary Analysis – Excel Deep Dive Project

## Overview
This **Salary Analysis** project is the advanced capstone from Luke Barousse's *Excel for Data Analytics* course.  
While Project 1 focuses on interactive visualization, this one dives deeper into **statistical analysis**, data preparation, and storytelling using the same 2023 data jobs dataset.

Goal: Uncover actionable insights about the data job market — top paying roles, skill correlations, location impact, experience premiums, and more — all in pure Excel.

## Key Analyses Performed
- Salary distribution by job title (median, quartiles, outliers)
- Top 10 highest-paying job titles and companies
- Salary comparison: Remote vs On-site vs Hybrid
- Experience level impact on compensation
- Country/region salary benchmarks (focus on high-volume markets)
- Basic what-if analysis (e.g., expected salary uplift with certain skills)
- Conditional formatting to highlight trends and anomalies

## Skills Demonstrated
- Advanced Power Query: Complex transformations, merging datasets, custom columns
- Formulas & functions: XLOOKUP, FILTER, UNIQUE, AVERAGEIFS, PERCENTILE, etc.
- Power Pivot / DAX: Measures for dynamic calculations (e.g., median salary by filters)
- Statistical summary tables and charts
- Data storytelling: Clear annotated visuals + insight write-ups
