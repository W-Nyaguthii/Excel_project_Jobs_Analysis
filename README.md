Data Jobs Salary Dashboard – Excel Interactive Project


## Overview
This interactive **Salary Dashboard** was built as the capstone project for Luke Barousse's *Excel for Data Analytics – Full Course for Beginners*.  
It helps job seekers, recruiters, and aspiring data professionals quickly explore real-world salary data for data science, data analyst, data engineer, and machine learning roles (based on 2023 job postings).

The dashboard lets users filter by job title, location (country/region), experience level, employment type, and more — revealing median salaries, salary distributions, and top-paying combinations at a glance.

## Key Features
- **Interactive Slicers** for job title, country, work setting (remote/hybrid/on-site), and experience level
- **Dynamic charts**: Bar charts for average salary by job title, box plots for salary ranges, pie charts for job distribution
- **Key metrics cards**: Median salary, average salary, total job postings, highest/lowest paying roles
- Clean layout with conditional formatting and sparklines for quick insights
- Built entirely in **Excel**
